# ve_radio

A Minetest mod that adds a radio block which randomly plays music files from the `sounds` folder inside the mod directory.

## Features

- Simple radio node that plays background music
- Randomly selects tracks from the music folder
- Easy to customize with your own music files

## Installation

1. Download or clone this mod into your Minetest mods directory:


2. Ensure the mod is enabled in your world configuration

3. Add your music files to the `sounds` folder inside the mod directory

## Music Files

Place your music files in the `sounds` folder within the mod directory.

**Supported file format:** `.ogg` (Ogg Vorbis)

Minetest's sound system requires Ogg Vorbis format for audio files. Make sure all your music files have the `.ogg` extension.

## Usage

1. Craft or obtain the ve_radio node through creative mode or crafting recipe
2. Place the radio block in your world
3. The radio will automatically start playing random tracks from the music folder
4. Right-click the radio to toggle it on/off (if implemented)
